Project Name: Hangman Game

Description:
This is a text-based Hangman game developed using Python. The player guesses a hidden word one letter at a time. The game ends when the player correctly guesses the word or uses all 6 attempts.

Features:
- Random word selection
- Letter-by-letter guessing
- 6 incorrect guess limit
- Input validation

Concepts Used:
- Python random module
- While loops
- If-else statements
- Lists
- Strings
- User input/output
